# Lovable Integration

This guide walks you through setting up a simple frontend interface for interacting with your Toolhouse agent.

> 💡 This guide assumes you already have an agent created. If you don’t, click this link to create one: [Example Agent - Search X and get email digest](https://app.toolhouse.ai/agents/new?message=i+want+an+agent+that+searches+x+for+something+and+sends+an+email+with+the+recap+of+the+search)

#### 1. Publish Your Agent

Once your agent is ready, click the **Publish** button. You’ll be taken to a screen that looks like this:

<figure><img src="https://3896932452-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fio16js3ONksvIYTf7YUl%2Fuploads%2FW9RLEJEPBtWPHcVg5txr%2FScreenshot%202025-08-03%20at%201.21.54%20PM.png?alt=media&#x26;token=44bad10a-14a8-4d11-93aa-bcfa8d651d3c" alt=""><figcaption></figcaption></figure>

#### 2. Get the Lovable Prompt

After publishing, navigate to the **Lovable** tab to get a pre-generated prompt. This prompt is automatically configured to create a chat interface that integrates with your Toolhouse agent.

This is what you’ll paste into [lovable.dev](https://lovable.dev/).

Your screen will look something like this:

<figure><img src="https://3896932452-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fio16js3ONksvIYTf7YUl%2Fuploads%2FWzNlQaroYdIw827s92c2%2FScreenshot%202025-08-03%20at%201.22.35%20PM.png?alt=media&#x26;token=dcf6fefd-7740-4266-8240-ce28bdf8a4d1" alt=""><figcaption></figcaption></figure>

#### 3. Go to Lovable.dev

Open [lovable.dev](https://lovable.dev/) in your browser.

You’ll land on a page with an input field like this:

<figure><img src="https://3896932452-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fio16js3ONksvIYTf7YUl%2Fuploads%2F7Q9RLMDCROgVlIpe8s80%2FScreenshot%202025-08-03%20at%201.23.38%20PM.png?alt=media&#x26;token=0f830753-ac0c-48c9-b24e-617f3d5d637b" alt=""><figcaption></figcaption></figure>

#### 4. Paste the Prompt

Paste the copied prompt into the input field.

Press **Enter**, and your frontend will be ready to interact with the agent.

<figure><img src="https://3896932452-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fio16js3ONksvIYTf7YUl%2Fuploads%2FQeE0P90GirxPKdX3gOxf%2FScreenshot%202025-08-03%20at%201.24.05%20PM.png?alt=media&#x26;token=8322ce70-a8fd-410f-afc1-d1e90687dae8" alt=""><figcaption></figcaption></figure>

You’ve now connected your Toolhouse agent to a frontend using Lovable! From here, you can use Lovable to further customize your app.

Need help? Visit [Toolhouse Support](http://help.toolhouse.ai/) or ask in our [Community Discord](https://discord.gg/2Pfsz6b6).
